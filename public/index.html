Webauthn UI and PaymentRequest APIs

This page shows an issue with Webauthn UI in context of PaymentRequest.show() call

Webauthn UI works well

in stand-alone page
  • Goto /pay/checkout
  • Click signUp button
  • Chrome opens Webauthn UI ('Verify your identity with...')

Webauthn UI doesn't work when it used inside PaymentRequest window in chrome-for-desktop, but it works with e.g. chrome-for-android

  • Press to open PaymentRequest UI
  • Click signUp button in the opened window
  • Expected: Chrome shows Webauthn UI
    Actual : Chrome doesn't show Webauthn UI
